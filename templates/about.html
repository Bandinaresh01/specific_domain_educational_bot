About Our Educational Chatbot

Overview

This AI-powered chatbot is designed to assist students with university syllabus queries. It uses advanced technologies to provide accurate, context-specific answers drawn directly from educational PDFs, making learning more efficient and engaging.

Features

PDF-based QA System

Answers retrieved from syllabus documents for precise, relevant information.

FAISS Vector Store

Fast similarity search for quick and scalable document retrieval.

Gemini AI Integration

Generates natural language responses using Google's advanced LLM.

Subject-wise Organization

Navigate and chat by specific subjects like Computer Networks, Data Structures, and more.

Tech Stack

Backend
Gemini 1.5 Flash
Embeddings
SentenceTransformer (SBERT)
Vector Store
FAISS
Web Framework
Flask (Python)
PDF Processing
